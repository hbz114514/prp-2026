🔭 望远镜指向仿真项目阶段性成果说明

###

🚀 快速启动流程
下载星表数据
运行 datadownload.py。这个脚本会从 ESA Gaia 数据库获取恒星信息，由于数据量较大，整个过程可能持续 10 分钟左右，请耐心等待直到生成 CSV 文件。

##

启动可视化界面

首先运行 integrated_server.py 以启动本地 API 服务。

然后在浏览器中打开 test.html。
你可以在网页上通过滑块实时调节方位角、仰角和焦距，查看 3D 模型与实时模拟星图的联动。

integrated_server.py & test.html
这是目前的核心成果，实现了望远镜姿态仿真、坐标转换、机械误差注入以及实时星图渲染。

image_generator_v2.py
这是一个手动输入参数返回图片的中间产物，目前主要用于调试逻辑。通常情况下可以忽略此文件。

dataset_generator.ipynb
算法开发专用。该脚本会正态生成随机的相机误差参数，自动生成一批“纯净”的测试星图（仅保留 星等 < 6.5 的亮星），去除了所有网格和标注，专门为下一步的“三角形匹配识别算法”提供训练和测试数据。我们固定了一个特定时间，并不是随机而是按照固定的圆轨道拍摄一系列图片，图片放在了压缩包中。同时，误差参数计算出矩阵，存入true_params.json供比对。

data_processor.py 处理这些星图，讲图片的坐标等数据提取出来放入csv文件，等待下一步的处理。

最终，final_solver.py读取这个csv，解出最合适的误差参数，并与.json文件比对误差。
你可以仅下载.csv与.json和final_solver.py进行调试，也可以调节dataset_generator中的随机种子生成图片再生成csv，测试算法对不同参数的鲁棒性。
